# Frozen Stacks

## Overview

This document records the current standard frozen stack versions supported by the OGS Launcher. These versions are distributed through the OGS frozen stacks repository and referenced by the remote mirror configuration.

## Standard Profile (v1.0)

- Godot: 4.3 (stable)
- Blender: 4.5.7 (stable)
- Krita: 5.2.15 (stable)
- Audacity: 3.7.7 (stable)

- Repo: https://github.com/OpenGameStack-Org/ogs-frozen-stacks
